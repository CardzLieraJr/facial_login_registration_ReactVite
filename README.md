# Facial Login and Registration

A modern facial login and registration user interface built with React, Vite, TypeScript, and Shadcn UI.

This project provides a clean authentication interface where users can register and log in using facial recognition features.

## Features

* Facial login interface
* Facial registration interface
* Responsive design
* Modern UI components
* Built with React and TypeScript
* Fast development setup using Vite
* Styled with Shadcn UI and Tailwind CSS

## Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS
* Shadcn UI
* PostCSS

## Getting Started

### Prerequisites

Make sure you have installed:

* Node.js version 18 or higher
* npm

### Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/facial-login-registration.git
```

Go to the project folder:

```bash
cd facial-login-registration
```

Install dependencies:

```bash
npm install
```

### Run the Project

Start the development server:

```bash
npm run dev
```

Open the local URL shown in your terminal, usually:

```text
http://localhost:5173
```

## Build for Production

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```
